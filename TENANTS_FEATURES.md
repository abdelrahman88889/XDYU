# 👥 الميزات الجديدة لصفحة المستأجرين

## نظرة عامة
تم تحديث شامل لصفحة إدارة المستأجرين مع إضافة حقول وميزات متقدمة لتتبع الدفعات الشهرية تلقائياً.

---

## 📋 الحقول الجديدة المضافة

### 1. **رقم الهوية** (ID Number)
- حقل مطلوب لتمييز المستأجرين
- يُستخدم في التحقق من البيانات

### 2. **رقم الجوال** (Phone)
- معلومات التواصل الأساسية
- صيغة: 0501234567

### 3. **الجنسية** (Nationality)
- تحديد جنسية المستأجر
- مثال: سعودي، كويتي، إماراتي

### 4. **نوع الوحدة** (Unit Type)
- تحديد نوع الوحدة المستأجرة
- الخيارات:
  - شقة
  - فيلا
  - محل تجاري
  - مكتب

### 5. **نوع الدفع** (Payment Type)
- تحديد دورية الدفع
- الخيارات:
  - شهري
  - ربع سنوي
  - نصف سنوي
  - سنوي

### 6. **قيمة الإيجار** (Rent Value)
- المبلغ الأساسي للإيجار
- بوحدة ر.س

### 7. **الضريبة** (Tax)
- نسبة الضريبة المطبقة
- تُحسب على قيمة الإيجار تلقائياً

---

## 🔄 نظام حساب الدفعات الشهري

### كيفية العمل
1. عند إضافة مستأجر، تُدخل تاريخ البداية والنهاية
2. النظام يحسب عدد الدفعات الشهرية تلقائياً
3. تُنشأ دفعة لكل شهر بـ:
   - تاريخ الاستحقاق الشهري
   - قيمة الإيجار
   - مبلغ الضريبة
   - الإجمالي (إيجار + ضريبة)
   - حالة الدفع (قيد الانتظار / مدفوع)

### مثال عملي
```
إضافة مستأجر:
- الاسم: محمد أحمد
- تاريخ البداية: 2024-01-15
- تاريخ النهاية: 2024-12-15
- قيمة الإيجار: 3000 ر.س
- الضريبة: 5%

النتيجة: 12 دفعة شهرية
- كل دفعة: 3000 + 150 ضريبة = 3150 ر.س
- مواعيد الاستحقاق: 15 من كل شهر
```

## 📊 جدول الدفعات التفاعلي

### المعلومات المعروضة
| العنصر | الوصف |
|------|-------|
| الشهر | اسم الشهر بالعربية والسنة |
| تاريخ الاستحقاق | التاريخ المحدد للدفع |
| الإيجار | المبلغ الأساسي |
| الضريبة | نسبة الضريبة المحسوبة |
| الإجمالي | المبلغ النهائي |
| الحالة | قيد الانتظار / مدفوع |
| الإجراء | زر تحصيل أو تاريخ الدفع |

### ملخص الدفعات
عند فتح تفاصيل المستأجر، يظهر:
- **إجمالي المبلغ**: مجموع جميع الدفعات
- **المدفوع**: المبلغ الذي تم تحصيله
- **المتبقي**: المبلغ المتبقي

---

## 🎯 الميزات التفاعلية

### 1. إضافة مستأجر جديد
```
الخطوات:
1. اضغط "+ إضافة مستأجر جديد"
2. ملء جميع الحقول المطلوبة (*)
3. النظام يحسب الدفعات تلقائياً
4. اضغط "حفظ المستأجر"
```

### 2. عرض تفاصيل المستأجر
```
الخطوات:
1. في الجدول الرئيسي، اضغط "📋 التفاصيل"
2. سيظهر جدول الدفعات الشهري
3. يعرض ملخص الدفعات والحالة
```

### 3. تحصيل الدفعات
```
الخطوات:
1. اضغط "✓ تحصيل" بجانب الدفعة
2. تُحدث حالة الدفعة إلى "مدفوع"
3. يُسجل تاريخ الدفع تلقائياً
4. يُحدث ملخص الدفعات
```

### 4. حذف مستأجر
```
الخطوات:
1. اضغط "حذف" في الصف
2. يُحذف المستأجر وجميع دفعاته
```

## 📱 التصميم المتجاوب

### على أجهزة الحاسوب
- عرض كامل الجدول مع التفاصيل
- تخطيط شبكي متقدم للنماذج
- ملخص الدفعات بـ 3 أعمدة

### على الأجهزة المحمولة
- جدول مضغوط مع ألوان مميزة
- نموذج عمود واحد
- ملخص الدفعات عمود واحد
- أزرار كاملة العرض

## 💾 هيكل البيانات

### كائن المستأجر الجديد
```javascript
{
  id: number,                    // رقم فريد
  name: string,                  // اسم المستأجر
  idNumber: string,              // رقم الهوية
  phone: string,                 // رقم الجوال
  nationality: string,           // الجنسية
  unit: string,                  // رقم الوحدة
  unitType: string,              // نوع الوحدة
  rentalStart: string,           // تاريخ البداية (YYYY-MM-DD)
  rentalEnd: string,             // تاريخ النهاية (YYYY-MM-DD)
  rentValue: number,             // قيمة الإيجار
  tax: number,                   // نسبة الضريبة %
  paymentType: string,           // نوع الدفع
  status: string,                // حالة المستأجر
  payments: [                    // مصفوفة الدفعات
    {
      month: string,             // شهر الدفعة (YYYY-MM-DD)
      dueDate: string,           // تاريخ الاستحقاق
      rentAmount: number,        // قيمة الإيجار
      taxAmount: number,         // مبلغ الضريبة
      totalAmount: number,       // المبلغ الإجمالي
      status: string,            // حالة الدفعة
      paymentDate: string|null   // تاريخ الدفع
    }
  ]
}
```

## ✨ الدوال الجديدة

### calculateMonthlyPayments()
**الوصف**: تحسب جميع الدفعات الشهرية من تاريخ البداية إلى النهاية

**المدخلات**:
- startDate: تاريخ البداية
- endDate: تاريخ النهاية
- rentValue: قيمة الإيجار
- tax: نسبة الضريبة (اختياري)

**المخرجات**: مصفوفة الدفعات الشهرية

### getArabicMonthName()
**الوصف**: تحول التاريخ إلى اسم الشهر بالعربية

**مثال**: "2024-01-15" → "يناير 2024"

## 🔍 نصائح الاستخدام

### ✅ أفضل الممارسات
- تأكد من إدخال تاريخ نهاية أكبر من تاريخ البداية
- استخدم تاريخ محدد للدفعات (مثلاً 15 من كل شهر)
- ضع نسبة ضريبة صحيحة بناءً على النظام المحلي
- حدث حالة الدفعات بانتظام

### ⚠️ ملاحظات مهمة
- الضريبة تُطبق على قيمة الإيجار فقط
- التاريخ يُحدد موعد الاستحقاق الشهري
- حذف المستأجر يحذف جميع دفعاته
- البيانات تُحفظ محلياً في المتصفح

## 🐛 استكشاف الأخطاء

### المشكلة: الدفعات لم تُحسب
**الحل**: تأكد من إدخال:
- تاريخ البداية والنهاية
- قيمة الإيجار بشكل صحيح

### المشكلة: ظهور قيم غريبة في الضريبة
**الحل**: نسبة الضريبة يجب أن تكون بين 0-100

### المشكلة: الدفعات لا تُحفظ
**الحل**: تأكد من وجود مساحة في localStorage
