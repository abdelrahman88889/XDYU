## 🔄 نظام الدفعات الذكي

### خطوات العمل
```
1. ملء حقول الإيجار والضريبة
2. اختيار التاريخ من والى
3. النظام ينشئ الدفعات تلقائياً
4. كل شهر = دفعة واحدة
5. تحسب الضريبة تلقائياً
6. عرض جدول شامل
7. تتبع التحصيل
```

### مثال على الحسابات
```
الإيجار:           3000 ر.س
نسبة الضريبة:      5%
قيمة الضريبة:      150 ر.س
دفعة واحدة:        3150 ر.س

للـ 12 شهر:
الإجمالي:          37,800 ر.س
المجموع المسدد:    0 ر.س (في البداية)
المتبقي:           37,800 ر.س
```
