## الأخطاء والحلول

### في حالة عدم التشغيل
1. تأكد من تثبيت Node.js
2. امسح node_modules: `rm -rf node_modules`
3. أعد التثبيت: `npm install`

### مشاكل الأداء
1. امسح cache: `npm cache clean --force`
2. جرب البناء: `npm run build`
3. تحقق من الذاكرة المتاحة
