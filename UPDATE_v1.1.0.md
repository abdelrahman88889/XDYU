## 🚀 كيفية الاستخدام

### إضافة مستأجر جديد
```
1. اضغط "+ إضافة مستأجر جديد"
2. ملء الحقول الجديدة:
   - الهوية
   - الجنسية
   - نوع الوحدة
   - نوع الدفع
   - قيمة الإيجار
   - الضريبة %
3. اضغط "حفظ المستأجر"
4. النظام يحسب الدفعات تلقائياً
```

### عرض جدول الدفعات
```
1. اضغط "📋 التفاصيل" للمستأجر
2. سيظهر جدول الدفعات الشهري
3. يعرض ملخص الدفعات
4. يمكن تحصيل كل دفعة بحدة
```

## 🔧 تفاصيل التطوير

### الدوال الجديدة

#### calculateMonthlyPayments()
```javascript
// الاستخدام
const payments = calculateMonthlyPayments(
  '2024-01-15',  // تاريخ البداية
  '2024-12-15',  // تاريخ النهاية
  3000,          // قيمة الإيجار
  5              // نسبة الضريبة %
);

// النتيجة: مصفوفة 12 دفعة شهرية
```

#### getArabicMonthName()
```javascript
// الاستخدام
const monthName = getArabicMonthName('2024-01-15');
// النتيجة: "يناير 2024"
```
