# تكامل Firebase - دليل شامل

## ✅ ما تم إنجازه

تم تحديث تطبيق إدارة العقارات بالكامل للعمل مع **Firebase**:

### 1. **المصادقة (Authentication)**
- ✓ تسجيل دخول عبر **Google**
- ✓ تسجيل دخول عبر **Facebook**
- ✓ تسجيل دخول عبر **Apple**
- ✓ تسجيل دخول عبر **Microsoft**
- ✓ تسجيل دخول عبر البريد الإلكتروني وكلمة المرور
- ✓ وضع "ضيف" (بدون حساب)

### 2. **قاعدة البيانات (Firestore)**
تم إنشاء مجموعات البيانات التالية:
- **users**: بيانات المستخدمين
- **tenants**: بيانات المستأجرين
- **units**: بيانات الوحدات
- **expenses**: بيانات المصروفات
- **payments**: بيانات الدفعات

### 3. **الملفات المحدثة**

#### `src/firebase.js`
```javascript
- تهيئة Firebase مع جميع المكتبات المطلوبة
- إضافة Authentication و Firestore
- إعداد persistence للجلسات
```

#### `src/firebaseService.js` (ملف جديد)
```javascript
- دوال التعامل مع Firestore (إضافة، تحديث، حذف، قراءة)
- real-time listeners للبيانات المحدثة تلقائياً
- دوال للعمل مع كل مجموعة بيانات
```

#### `src/pages/Login.js`
```javascript
- تسجيل دخول حقيقي عبر Firebase
- معالجة أخطاء المصادقة
- إنشاء حسابات جديدة
- تسجيل بيانات المستخدم في Firestore
```

#### `src/App.js`
```javascript
- مراقبة حالة المصادقة (Firebase Auth)
- تسجيل الخروج الآمن
- عرض اسم المستخدم في الواجهة
```

#### `src/pages/Dashboard.js`
```javascript
- تحميل البيانات من Firestore بدلاً من localStorage
- real-time updates للإحصائيات
```

#### `src/pages/Tenants.js`, `Units.js`, `Collections.js`, `Expenses.js`
```javascript
- استخدام firebaseService للعمليات
- دعم كامل للمستخدمين المسجلين والضيوف
- حفظ بيانات في Firestore للمستخدمين المسجلين
- حفظ بيانات في localStorage للضيوف
```

## 🚀 كيفية الاستخدام

### للمستخدمين المسجلين (عبر Firebase):

1. **قم بالدخول إلى الموقع**
2. **اختر طريقة الدخول:**
   - 🔍 Google
   - 🍎 Apple
   - f Facebook
   - ⊞ Microsoft
   - ✉️ البريد الإلكتروني وكلمة المرور

3. **أدخل بيانات اعتمادك الحقيقية**
4. **سيتم حفظ جميع بياناتك في Firebase تلقائياً**

### للضيوف:

1. **اضغط "متابعة كضيف"**
2. **ستتمكن من الوصول لجميع الميزات**
3. **البيانات ستُحفظ في localStorage (الجهاز فقط)**

## 🚨 استكشاف الأخطاء

### إذا لم يعمل تسجيل الدخول عبر Google:
- تأكد من تفعيل Google Sign-In في Firebase Console
- تأكد من السماح بـ localhost في OAuth settings

### إذا لم تظهر البيانات:
- تحقق من اتصالك بالإنترنت
- تحقق من قواعد Firestore Security Rules

### إذا حدثت مشاكل في الأداء:
- قد تحتاج إلى تحسين فهرسة Firestore
- تحقق من عدد الـ queries الكبيرة
