## 🚀 طريقة الاستخدام

### 1. إضافة مستأجر جديد
```
1. اضغط "+ إضافة مستأجر جديد"
2. ملء جميع الحقول (المطلوبة مع *)
3. اضغط "حفظ المستأجر"
→ النظام ينشئ الدفعات تلقائياً
```

### 2. عرض الدفعات
```
1. اضغط "📋 التفاصيل"
2. يظهر جدول الدفعات الشهري
3. يعرض الملخص والإحصائيات
```

### 3. تحصيل دفعة
```
1. اضغط "✓ تحصيل" بجانب الدفعة
2. تُحدث الحالة إلى "مدفوع"
3. يُسجل التاريخ تلقائياً
4. يُحدث الملخص
```
