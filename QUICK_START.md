## 🚀 البدء السريع

### 1. إضافة مستأجر جديد
```
الخطوات:
1. انقر "+ إضافة مستأجر جديد"
2. ملء البيانات الأساسية
3. حدد نوع الدفع
4. انقر "✓ إضافة"
```

### 2. عرض جدول الدفعات
```
الخطوات:
1. ابحث عن المستأجر في الجدول
2. انقر "📅 الدفعات"
3. عدّل التواريخ إذا لزم الأمر
4. حدد الدفعات المدفوعة
```

### 3. تتبع التحصيلات
```
الخطوات:
1. اذهب لصفحة "التحصيلات"
2. شاهد الرسوم البيانية
3. حدد الدفعات المدفوعة
4. راقب شريط التقدم
```

## 🔍 نصائح الاستخدام

### ✨ النصيحة 1: جدول الدفعات
- اضغط على تاريخ الدفعة مباشرة لتعديله
- اضغط خارج الحقل لحفظ التعديل

### ✨ النصيحة 2: البحث عن المتأخرين
- شاهد عمود "متأخر" في جدول المستأجرين
- الأرقام الحمراء تشير للتأخيرات

### ✨ النصيحة 3: الرسوم البيانية
- حرّك الماوس على الأعمدة لرؤية التفاصيل
- قارن بين الرسم الشهري والسنوي

### ✨ النصيحة 4: التحصيل
- سجّل الدفعات فوراً بعد استلامها
- شاهد النسبة في شريط التقدم
