# تطبيق إدارة العقارات - دليل شامل

## نظرة عامة
تطبيق React متكامل لإدارة العقارات والمستأجرين والتحصيل. يوفر واجهة عربية سهلة الاستخدام مع جميع الميزات الأساسية والمتقدمة.

## المتطلبات
- Node.js 14.0 أو أحدث
- npm 6.0 أو أحدث
- متصفح حديث يدعم ES6

## التثبيت والتشغيل

### 1. تثبيت المكتبات
```bash
npm install
```

### 2. تشغيل التطبيق في وضع التطوير
```bash
npm start
```
سيفتح التطبيق على العنوان: http://localhost:3000

### 3. بناء التطبيق للإنتاج
```bash
npm run build
```

### 4. تشغيل الاختبارات
```bash
npm test
```

## المميزات الرئيسية

### 1. لوحة التحكم (Dashboard)
- عرض إحصائيات العقارات والمستأجرين
- معدل إشغال الوحدات
- إجمالي التحصيل والتحصيل المتوقع
- ملخص العمليات الأخيرة

### 2. إدارة المستأجرين (Tenants)
- إضافة مستأجرين جدد
- عرض قائمة المستأجرين
- حذف المستأجرين
- تتبع بيانات الاستئجار

### 3. إدارة الوحدات (Units)
- إضافة وحدات جديدة (شقق، فلل، محلات تجارية)
- عرض حالة الوحدات (مأجورة/فارغة)
- معلومات المستأجرين الحاليين
- تتبع أسعار الإيجار

### 4. إدارة التحصيل (Collections)
- تسجيل الإيجارات المستحقة
- تتبع الدفعات المدفوعة والمتأخرة
- عرض ملخص التحصيل
- وضع علامة على الدفعات المدفوعة

## الميزات الإضافية

### أدوات مساعدة (utils.js)
- تنسيق العملات
- تنسيق التواريخ بالعربية
- التحقق من البيانات (الهاتف، البريد الإلكتروني)
- حسابات معدل الإشغال
- إدارة التخزين المحلي

### التكوين (config.js)
- الألوان والأنماط الموحدة
- رسائل الخطأ والنجاح
- قائمات الاختيارات
- قواعد التحقق من الصحة
- تفاصيل التطبيق

## استخدام البيانات

### التخزين المحلي
البيانات تُحفظ في localStorage متصفح الويب. يمكن تصديرها واستيرادها باستخدام دوال utils.

### هيكل البيانات

#### الوحدات (Units)
```javascript
{
  id: number,
  number: string,
  type: string,
  location: string,
  area: number,
  rentPrice: number,
  status: 'مأجور' | 'فارغ',
  tenant: string
}
```

#### المستأجرين (Tenants)
```javascript
{
  id: number,
  name: string,
  phone: string,
  unit: string,
  rentalStart: string,
  rentalEnd: string,
  status: 'نشط' | 'غير نشط'
}
```

#### التحصيل (Collections)
```javascript
{
  id: number,
  unit: string,
  tenant: string,
  amount: number,
  month: string,
  status: 'مدفوع' | 'متأخر' | 'قيد الانتظار',
  paymentDate: string
}
```

## تطوير إضافي

### إضافة ميزات جديدة
1. أنشئ مكون جديد في `src/pages/`
2. أضف الطريقة في `src/App.js` بـ Router
3. أضف الملاحة في الـ navbar

### تعديل الأنماط
- عدّل `src/App.css` للتصميم العام
- عدّل `src/index.css` للأنماط الأساسية

### إضافة دوال مساعدة
- أضف الدوال الجديدة في `src/utils.js`
- استخدم الإعدادات من `src/config.js`

## استكشاف الأخطاء

### التطبيق لا يعمل
```bash
# تأكد من تثبيت المكتبات
npm install

# امسح node_modules وثبت من جديد
rm -rf node_modules
npm install

# امسح cache npm
npm cache clean --force
```

### مشاكل في التشغيل
- تأكد من أن العنوان 3000 غير مستخدم
- تحقق من إصدار Node.js: `node --version`
- جرب `npm start` من جديد
