## 🚀 كيفية الاستخدام

### الخطوة 1: شغّل التطبيق
```bash
npm start
```

### الخطوة 2: أضف مستأجر جديد
```
1. اضغط "+ إضافة مستأجر جديد"
2. ملء الحقول (المطلوبة مع *)
3. أدخل رقم العقد: CNT-2024-001
4. أدخل رقم السند: DOC-2024-001
5. اضغط "حفظ المستأجر"
```

### الخطوة 3: شاهد الإحصائيات
```
1. اذهب لصفحة التحصيل (💰)
2. شاهد الإحصائيات الجديدة في الأسفل
3. ستجد تحصيل الشهر والسنة
```
