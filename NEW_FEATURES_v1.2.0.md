## 🆕 الحقول المضافة

### 1. رقم العقد (Contract Number)
```javascript
contractNumber: "CNT-001"
```
**الغرض**: تسجيل رقم العقد الخاص بالمستأجر
**الموقع**: نموذج إضافة مستأجر جديد
**الإجراء**: مطلوب (*)
**مثال**: CNT-001, CNT-2024-001, etc.

### 2. رقم السند (Document Number)
```javascript
documentNumber: "DOC-001"
```
**الغرض**: تسجيل رقم السند أو الوثيقة
**الموقع**: نموذج إضافة مستأجر جديد
**الإجراء**: مطلوب (*)
**مثال**: DOC-001, DOC-2024-001, etc.

**مثال العرض**:

## 🔄 التغييرات في صفحة المستأجرين

### جدول المستأجرين - أعمدة جديدة

**الأعمدة السابقة**:
```
الاسم | الهوية | الجوال | الجنسية | الوحدة | الإيجار | الدفعات | الحالة | الإجراءات
```

**الأعمدة الجديدة**:
```
الاسم | الهوية | الجوال | رقم العقد | رقم السند | الوحدة | الإيجار | الدفعات | الحالة | الإجراءات
```

**التحسين**: 
- إزالة عمود الجنسية من الجدول الرئيسي
- إضافة عمودي رقم العقد ورقم السند
- تنظيم أفضل للمعلومات المهمة
