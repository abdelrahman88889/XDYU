## ✨ ما الجديد؟

### 2️⃣ حقول جديدة
```
✅ رقم العقد (Contract Number)
   مثال: CNT-2024-001
   
✅ رقم السند (Document Number)
   مثال: DOC-2024-001
```

### 📊 إحصائيات جديدة
```
✅ تحصيل الشهر الحالي
   يعرض: كم تم تحصيله، الإجمالي المتوقع
   
✅ تحصيل السنة كاملة
   يعرض: كم تم تحصيله، الإجمالي المتوقع
```

---

## 📍 أين الإضافات؟

### صفحة المستأجرين
```
نموذج إضافة مستأجر:
- اضغط "+ إضافة مستأجر جديد"
- ستجد حقلين جديدين:
  ✅ رقم العقد *
  ✅ رقم السند *

جدول المستأجرين:
- الأعمدة الجديدة:
  ✅ رقم العقد (بين الجوال والوحدة)
  ✅ رقم السند
```

### صفحة التحصيل (Collections)
```
في الأعلى تحت الملخص العام:
📊 إحصائيات التحصيل (قسم جديد)
├─ تحصيل الشهر الحالي: XXX ر.س من YYY ر.س
└─ تحصيل السنة كاملة: XXX ر.س من YYY ر.س
```

---

## 🚀 الاستخدام

### إضافة مستأجر جديد
```
1. اضغط "+ إضافة مستأجر جديد"
2. ملء جميع الحقول (المطلوبة مع *)
3. أدخل:
   - رقم العقد: CNT-2024-001
   - رقم السند: DOC-2024-001
4. اضغط "حفظ المستأجر"
```

### عرض الإحصائيات
```
1. اذهب لصفحة التحصيل
2. شاهد القسم الأخضر الجديد:
   📊 إحصائيات التحصيل
3. ستجد:
   - تحصيل الشهر الحالي
   - تحصيل السنة كاملة
```
