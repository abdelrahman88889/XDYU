## 🔐 اختبارات تسجيل الدخول والمصادقة

### اختبار 1: إنشاء حساب جديد
```
الخطوة: أدخل بريد إلكتروني جديد: user1@test.com
الخطوة: أدخل كلمة مرور: password123
الخطوة: انقر "إنشاء الحساب"
المتوقع: ✅ يجب الدخول إلى Dashboard فوراً
التحقق: هل يظهر البريد الإلكتروني في الرأس؟
```

### اختبار 2: تسجيل الدخول بحساب موجود
```
الخطوة: استخدم نفس البريد: user1@test.com
الخطوة: أدخل كلمة المرور الصحيحة
الخطوة: انقر "تسجيل الدخول"
المتوقع: ✅ يجب الدخول إلى Dashboard
التحقق: هل البيانات السابقة موجودة؟
```

### اختبار 3: كلمة مرور خاطئة
```
الخطوة: استخدم نفس البريد: user1@test.com
الخطوة: أدخل كلمة مرور خاطئة
الخطوة: انقر "تسجيل الدخول"
المتوقع: ❌ يجب ظهور رسالة خطأ
الخطأ المتوقع: "البريد الإلكتروني أو كلمة المرور غير صحيحة"
```

### اختبار 4: خيار المتابعة كضيف
```
الخطوة: انقر "متابعة كضيف"
المتوقع: ✅ يجب الدخول إلى Dashboard كضيف
التحقق: هل يظهر "أنت تتصفح كضيف"؟
```

الخطوة: انظر إلى القسم السفلي

التحقق: 
- هل الرسوم البيانية تظهر بشكل صحيح؟
- هل الألوان صحيحة (بنفسجي)؟

## 👥 اختبارات إضافة المستأجرين

### اختبار 7: إضافة مستأجر أول
```
مستخدم: user1@test.com
الخطوة: اذهب إلى "المستأجرين"
الخطوة: انقر "+ إضافة مستأجر جديد"
الخطوة: ملء الحقول:
- الاسم: أحمد محمد
- رقم الهوية: 1234567890
- الهاتف: 0501234567
- الجنسية: سعودي
- الوحدة: الفيلا 1
- نوع الوحدة: فيلا
- رقم العقد: CNT-001
- رقم الوثيقة: DOC-001
- تاريخ البداية: 2026-01-01
- تاريخ النهاية: 2027-01-01
- الإيجار: 3000
- الضريبة: 5%
- نوع الدفع: شهري
- الحالة: نشط

الخطوة: انقر "إضافة"
المتوقع: ✅ يجب إضافة المستأجر والظهور في الجدول
```

### اختبار 8: عرض جدول الدفعات
```
الخطوة: انقر "عرض الدفعات" للمستأجر المضاف
المتوقع: ✅ جدول به 12 دفعة شهرية
التحقق:
- هل كل الدفعات موجودة؟
- هل الحالة = "قادم"؟
- هل المبلغ = 3150 (3000 + 5%)؟
```

### اختبار 9: تعديل تاريخ الدفعة
```
الخطوة: انقر على تاريخ أول دفعة
الخطوة: غيّر التاريخ إلى: 2026-02-01
الخطوة: احفظ (انقر خارج الحقل)
المتوقع: ✅ التاريخ يجب أن يتحدث فوراً
التحقق: هل التاريخ تحدث في الجدول؟
```

### اختبار 10: تحديد دفعة كمدفوعة
```
الخطوة: انقر "تم الدفع" لأول دفعة
المتوقع: ✅ الحالة تتغير إلى "مدفوع"
الخطوة: ادخل تاريخ الدفع الفعلي: 2026-01-15
المتوقع: ✅ تاريخ الدفع يظهر في الجدول
```

## 📊 اختبارات لوحة التحكم مع البيانات

### اختبار 11: تحديث الإحصائيات
```
ارجع إلى "لوحة التحكم" بعد إضافة المستأجر والدفعة
المتوقع: ✅ يجب تحديث الأرقام:
- عدد المستأجرين: 1
- المستأجرين النشطين: 1

- المبلغ المحصل: 3150 ر.س (1 دفعة)

- معدل التحصيل: ~8.4%
```

## 🔄 اختبارات تعدد المستخدمين

### اختبار 12: مستخدم ثاني مختلف تماماً
```
الخطوة: تسجيل الخروج من user1@test.com
الخطوة: انقر "تسجيل الخروج"
المتوقع: ✅ يجب العودة إلى صفحة Login

الخطوة: إنشاء حساب جديد: user2@test.com
الخطوة: ادخل كلمة مرور مختلفة: different123
الخطوة: انقر "إنشاء الحساب"

المتوقع: ✅ Dashboard فارغ تماماً
التحقق: 
- هل جميع الأرقام = 0؟
- هل لا توجد بيانات من user1؟
```

### اختبار 13: فصل البيانات بين المستخدمين
```
للمستخدم user2:
الخطوة: أضف مستأجر مختلف (اسم: علي سالم، إيجار: 5000)
الخطوة: أضف دفعة واحدة

الخطوة: تسجيل الخروج
الخطوة: سجّل دخول باستخدام user1@test.com

المتوقع: ✅ Dashboard يجب أن يظهر بيانات user1 الأصلية فقط
التحقق:
- الإيجار الكلي: 3000 (ليس 8000)
- المستأجر الموجود: أحمد محمد (ليس علي سالم)
```

## 💾 اختبارات حفظ البيانات

### اختبار 14: استمرار البيانات بعد التحديث
```
مستخدم: user1@test.com
الخطوة: اذهب إلى "المستأجرين"
الخطوة: لاحظ البيانات الحالية
الخطوة: أعد تحميل الصفحة (F5)

المتوقع: ✅ جميع البيانات موجودة بعد التحديث
التحقق:
- المستأجرين موجودين؟
- جدول الدفعات موجود؟
- التعديلات السابقة محفوظة؟
```

### اختبار 15: حفظ البيانات عند إغلاق المتصفح
```
الخطوة: أضف مستأجر جديد
الخطوة: احفظ البيانات
الخطوة: أغلق المتصفح تماماً
الخطوة: فتح المتصفح مرة أخرى
الخطوة: ادخل إلى الموقع

المتوقع: ✅ جميع البيانات موجودة
```
