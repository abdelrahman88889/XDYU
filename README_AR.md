# 🏢 نظام إدارة العقارات المتقدم

## النسخة 1.4.0 - النظام الكامل

### 🎯 الوصف
تطبيق ويب احترافي لإدارة العقارات السكنية والتجارية، مع نظام تسجيل دخول آمن، وإدارة المستأجرين، وتتبع الإيجارات والدفعات، مع رسوم بيانية متقدمة وإحصائيات شاملة.

---

## 🚀 البدء السريع

### 1️⃣ التثبيت
```bash
cd real-estate-app
npm install
```

### 2️⃣ التشغيل
```bash
npm start
```
ثم افتح المتصفح على: `http://localhost:3000`

### 3️⃣ البناء للإنتاج
```bash
npm run build
```
